Stock Simulator

This python program is capable of getting stock name and code from csv file from Nasdaq. It uses the csv file and put the stock symbols into MySQL DataBase. Then, the program uses python API to get OHLC(open, high, low, close) prices. The prices are inserted into the database as well. The ultimate goal is to develope a tensorflow-based machine learning model in order to build a quantitative analysis model. The code for this post is on my GitHub account. You are welcome to use and modify the code. Thank you. https://github.com/tjeong117/getStock I will put some parts of the codes to demonstrate the functionality and purpose of them. The code is devided into two classes. One is to get stock prices and the other is to display the graph.
